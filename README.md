# Plant with Code - Wiki

Welcome to the **Plant with Code** Wiki repository! 

This is the un-official wiki page for the Roblox game **Plant with Code [BETA]**, created by the Roblox group *Lated graham* (founded by TheEmreTutuk). The game is inspired by *Grow a Garden*, but features custom Luau coding mechanics.

## 🛠️ Built With

This project is built primarily using core web technologies to ensure it is lightweight, fast, and accessible. It emphasizes simplicity and fundamental frontend development:

*   **HTML5**: Used for structuring the content of the wiki pages.
*   **CSS3**: Used for all the styling, layouts, and visual design to make the wiki look great.
*   **JavaScript**: Used for interactive elements and dynamic functionalities within the wiki.

*Note: This project deliberately focuses on Vanilla HTML, CSS, and JS without heavy frameworks.*

## 🚀 Getting Started

To view or work on this wiki locally, simply clone the repository and open the `index.html` file in your preferred web browser. No complex build tools or package managers are required!

```bash
# Clone the repository
git clone https://github.com/your-username/your-repo-name.git

# Navigate to the project directory
cd your-repo-name

# Open index.html in your browser (example using xdg-open on Linux)
xdg-open index.html
```
